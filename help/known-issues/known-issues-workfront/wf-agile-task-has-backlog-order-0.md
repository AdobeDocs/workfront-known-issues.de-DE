# Agile: Aufgabe hat einen Kanban-Rückstand von 0

Wenn ein(e) Benutzende(r) den Rückstand eines Kanban-Teams aufruft, zeigt eine oder mehrere der Aufgaben einen Rückstand von 0 an.
