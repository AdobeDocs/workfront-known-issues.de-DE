# Agile: Aufgabe hat einen Kanban-Rückstau von 0

Wenn ein Benutzer den Rückstand eines Kanban-Teams anzeigt, zeigt eine oder mehrere der Aufgaben einen Rückstand von 0 an.

_Erste Meldung am 7. Oktober 2022._
