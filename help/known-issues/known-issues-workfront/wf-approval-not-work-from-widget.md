# Genehmigungen: Dokument kann nicht 端ber das Widget Meine Genehmigungen in der Startseite genehmigt werden

Wenn ein(e) Benutzende(r) versucht, ein Dokument 端ber das Widget Meine Genehmigungen in der Startseite zu genehmigen, verschwindet das Dokument aus dem Widget. Das Dokument wird jedoch nicht genehmigt. Wenn der/die Benutzende die Seite aktualisiert, wird das Dokument erneut im Widget angezeigt.
