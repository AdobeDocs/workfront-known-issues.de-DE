# Genehmigungen: Dokument kann nicht über das Widget Meine Genehmigungen in der Startseite genehmigt werden

Wenn ein(e) Benutzende(r) versucht, ein Dokument über das Widget Meine Genehmigungen in der Startseite zu genehmigen, verschwindet das Dokument aus dem Widget. Das Dokument wird jedoch nicht genehmigt. Wenn der/die Benutzende die Seite aktualisiert, wird das Dokument erneut im Widget angezeigt.

_Erste Meldung am 10. Februar 2026._
