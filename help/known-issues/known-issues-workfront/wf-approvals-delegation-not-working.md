# Genehmigungen: Delegierte Genehmigungen, die nicht fÃ¼r Delegierte erscheinen

Wenn ein Benutzer eine Genehmigung an einen anderen Benutzer delegiert, kann der delegierte Benutzer die Genehmigung nicht sehen und kann keine Genehmigungsentscheidung treffen.

_Erste Meldung am Samstag, 22. November 2024._
