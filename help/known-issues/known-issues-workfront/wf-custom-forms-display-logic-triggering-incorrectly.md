# Benutzerdefinierte Formulare: Die Anzeigelogik der Felder wird nicht korrekt befolgt

Wenn ein(e) Benutzende(r) ein benutzerdefiniertes Formular ausf체llt, das eine Anzeigelogik enth채lt (bei der einige Felder basierend auf den Werten in anderen Feldern angezeigt werden oder nicht), werden die Felder nicht auf eine Weise angezeigt, die dieser Feldlogik folgt.
