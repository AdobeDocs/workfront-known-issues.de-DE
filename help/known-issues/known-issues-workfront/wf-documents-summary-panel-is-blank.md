# Dokumente: Zusammenfassung Bedienfeld ist leer

Wenn ein User eine Dokumente Liste anzeigt und auf eine Dokument klickt, wird das Zusammenfassung Bedienfeld geöffnet, bleibt aber leer. Der User kann das Zusammenfassung Bedienfeld nicht schließen.

Wenn der User die Seite neu lädt, füllt sich das Zusammenfassung Bedienfeld wie erwartet. Wenn der User jedoch auf eine andere Dokument klickt, ist der Zusammenfassungsbereich dieses Dokument leer.

**Umgehungslösung**

Um das Zusammenfassung Bedienfeld vorübergehend zu Ansicht, laden Sie das Seite neu.

_Erste Meldung am 13. Januar 2025._
