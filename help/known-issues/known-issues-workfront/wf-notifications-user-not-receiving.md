# Benachrichtigungen: Der/die Benutzende erhält keine erwarteten Benachrichtigungen

Wenn ein Ereignis auftritt, bei dem eine Benachrichtigung an einen Benutzer gesendet werden soll, erhält dieser Benutzer die Benachrichtigung nicht. Dies kann auch dann vorkommen, wenn bei einer Benutzerbenachrichtigungseinstellung die Benachrichtigung zulässig wäre.

Dieses Problem betrifft sowohl In-App- als auch E-Mail-Benachrichtigungen.

_Erste Meldung am Freitag, 12. Februar 2026._
