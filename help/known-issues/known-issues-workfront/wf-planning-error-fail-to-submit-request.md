# Workfront-Planung: Fehler „Anfrage konnte nicht gesendet werden“

Wenn ein(e) Benutzende(r) eine Anfrage sendet, wird die Meldung „Anforderung kann nicht gesendet werden“ angezeigt. Die Anfrage wird jedoch korrekt übermittelt.

_Erste Meldung am 11. Februar 2026._
