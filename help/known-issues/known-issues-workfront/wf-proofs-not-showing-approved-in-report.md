# Testsendungen: Testsendungen werden in Berichten nicht als genehmigt angezeigt

>[!NOTE]
>
>Dieses Problem wurde am 21. Juli 2023 behoben.

Wenn ein Benutzer einen Bericht anzeigt, der Testversandvalidierungsentscheidungen enth채lt, zeigt der Bericht nicht die [!UICONTROL Genehmigt] Entscheidung f체r alle genehmigten Testsendungen. Wenn der Benutzer den Testversand selbst ansieht, kann er erkennen, dass er genehmigt wurde.
