# Testsendungen: Testsendungen verbleiben im Widget Meine Validierungen .

Ein Testversand, der aus dem Widget Meine Genehmigungen verschwinden sollte, verbleibt im Widget. Dies kann vorkommen, wenn mehrere Benutzer gleichzeitig Entscheidungen 체ber einen Testversand treffen oder ein Benutzer eine Entscheidung trifft und diese schnell 채ndert.

_Erste Meldung am Donnerstag, 7. Februar 2024._
