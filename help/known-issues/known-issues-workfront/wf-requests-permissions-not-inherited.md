# Anfragen: Berechtigungen werden beim Erstellen einer Anfrage nicht vererbt

>[!NOTE]
>
>Dieses Problem wurde am Samstag, 19. Dezember 2025 behoben.

Wenn ein Benutzer eine Anfrage erstellt, erbt diese Anfrage keine Berechtigungen. Dies bedeutet, dass Benutzende möglicherweise nicht auf die Anfrage zugreifen können.

_Erste Meldung am Freitag, 20. November 2025._
