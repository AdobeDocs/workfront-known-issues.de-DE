# Aufgaben: Im Dialogfeld „Bearbeiten“ einer Aufgabe vorgenommene Änderungen werden nicht gespeichert

>[!NOTE]
>
>* Dieses Problem betrifft das alte Aufgabenerlebnis.
>* Das Produkt-Team bewertet derzeit diese Problemlösung, die möglicherweise eine Produktverbesserung erfordert. Produktverbesserungen werden in den Produktankündigungen und nicht in den Wartungs-Updates kommuniziert.

Wenn Benutzende Änderungen am Abschnitt „Änderungen“ einer Aufgabe im älteren Dialogfeld für Aufgaben vornehmen und speichern werden die Änderungen nicht gespeichert.

**Umgehungslösung**

Für einige Änderungen können Sie das neue Aufgabenerlebnis verwenden.

_Erste Meldung am 24. November 2025._
