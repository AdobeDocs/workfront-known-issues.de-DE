# Aufgaben: Benutzerrolle kann beim Erstellen einer Aufgabe nicht ausgewählt werden

>[!NOTE]
>
>Dieses Problem wurde am 21. Dezember 2025 behoben.

Wenn Benutzende eine Aufgabe erstellen und versuchen, eine Benutzerrolle zuzuweisen, ist die Option zum Zuweisen der Rolle nicht vorhanden und die Person kann keine Benutzerrolle zuweisen.

**Umgehungslösung**

Erstellen Sie die Aufgabe und bearbeiten Sie dann die Aufgabe, um die Benutzerrolle in „Erweiterte Zuweisungen“ zuzuweisen.

_Erste Meldung am 28. November 2025._
