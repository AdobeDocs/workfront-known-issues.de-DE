# Aufgaben: Inline-Bearbeitungen an einer Aufgabe werden nicht gespeichert

>[!NOTE]
>
>Dieses Problem betrifft das alte Aufgabenerlebnis.

Wenn ein(e) Benutzende(r) im alten Aufgabenerlebnis Inline-Änderungen an einer Aufgabe vornimmt und speichert, werden die Änderungen nicht gespeichert.

**Umgehungslösung**

Für einige Änderungen können Sie das neue Aufgabenerlebnis verwenden.

_Erste Meldung am Freitag, 4. Dezember 2025._
