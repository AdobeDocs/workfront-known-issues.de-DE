# Aufgaben: Große Aufgabenliste springt unerwartet an eine andere Stelle in der Liste

Wenn ein(e) Benutzende(r) eine große Aufgabenliste öffnet und eine Aufgabe aktualisiert oder eine übergeordnete Aufgabe erweitert, springt die Aufgabenliste unerwartet an eine andere Stelle in der Liste.

_Erste Meldung am 22. Januar 2026._
