# Aufgaben: [!UICONTROL Vorgänger] Symbol fehlt in der Aufgabenkopfzeile

Wenn ein Benutzer eine Aufgabe anzeigt, fehlt das Symbol des Vorgängers der Aufgabe in der Kopfzeile.
