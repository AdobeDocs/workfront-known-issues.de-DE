# Benutzende: Benutzende werden in der Adobe Admin Console nicht korrekt bereitgestellt

Wenn eine Benutzerin oder ein Benutzer bereits auf einer Adobe Admin Console vorhanden ist und dann automatisch auf einer Konsole bereitgestellt wird, die Workfront enthält, wird diese Person nicht korrekt auf der neuen Konsole bereitgestellt.

_Erste Meldung am Freitag, 20. November 2025._
